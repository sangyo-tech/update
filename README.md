# update
産業技術アカデミーは、動画を提供して終わりの教育サービスではなく、 企業の教育を仕組み化し、継続的に教育レベルを向上させるための教育プラットフォームを目指しています。  このページでは、今後追加予定の機能やサービスの強化方針についてご紹介しています。  今後は、コンテンツの継続追加だけでなく、学習管理機能、認定証発行、質問・相談機能、AIによる学習支援など、 企業教育・個人学習の両方に役立つ仕組みを順次追加していく予定です。  産業技術アカデミーは、単なる動画学習サービスではなく、 製造業・技術系企業の教育インフラとなることを目標に、サービスの改善と機能追加を継続していきます。
